Get the Custom Uploader code to add in ShareX to be able to upload to this folder.

  1. Copy the below code to your clipboard.
  2. Open the ShareX window.
  3. Click the "Destinations" drop-down then click "Custom uploader settings...".
  4. Click the "Import" drop-down then click "From clipboard".
  5. Make sure the uploaders are set to Umbraco.
{
    "Version": "13.1.0",
    "Name": "Umbraco",
    "DestinationType": "ImageUploader, TextUploader, FileUploader",
    "RequestType": "POST",
    "RequestURL": "https://s-ul.eu/api/v1/upload",
    "Arguments": {
        "wizard": "true",
        "key": "mykey65498465416546546541"
    },
    "FileFormName": "file",
    "URL": "$json:url$",
    "ThumbnailURL": "$json:url$?thumb=1",
    "DeletionURL": "https://s-ul.eu/delete.php?key=mykey65498465416546546541&file=$json:filename$"
}
